BASH Tips

Apr 3, 2019 | 阅读

sudo !!

Force the system to check all file systems at boot:

sudo touch /forcefsck

Use cat to create or append text files

cat > myfile.txt
– Create small text file. Use Ctrl+D to send text to file.

cat >> myfile.txt
– Appends lines to bottom of pre_existing file.

Use pushd and popd to put move directories on and off a stack

pushed /directory
– Moves to new directory

popd
– brings you back to original directory

Open new root bash environment:

sudo bash
– Use “exit” to return to your own shell.

Use wget to download files from the internet:

wget https://....

Hide your cursor:

tput civis
– Hides cursor

tput cnorm

Fix a terminal that’s gone crazy:

. ~/.profile

. ~/.bashrc
– Reload your envornment.

reset
– resets the terminal completely.

Find a command in history:

Ctrl+R
history

String commands together at promt as they would run in a script:

command1; command2; command3; ...
